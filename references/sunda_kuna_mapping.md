# Sunda Kuna Master Reference Mapping

## Purpose
This file is the single authoritative reference for all Sunda Kuna vocabulary revisions across the novel "Persekutuan Keris." All revision agents must use these mappings consistently.

## Setting Context
Pre-Islamic 8th-9th century Nusantara (Hindu-Buddhist period). Modern Sundanese terms that entered via Islamic influence (Arabic loanwords) or later cultural developments must be replaced with attested Old Sundanese (Sunda Kuna) or Sanskrit-derived forms.

---

## A. Pronoun Replacements

| Modern (Remove) | Sunda Kuna (Use) | Context |
|---|---|---|
| Abdi (formal I) | **Ingwang** | All Sundanese formal dialogue |
| Aing (casual I, modern) | **Aing/Aming** | Keep — attested in Old Sundanese |
| Anjeun (formal you) | **Andika** | Formal address |
| Maneh (casual you) | **Dika** | Casual address |
| Manehna (he/she) | **Siya** | Third person |
| Urang (we, modern) | **Kami** | First person plural |

**Notes:**
- "Abdi" derives from Arabic "abd"/"abdul" via Islamic Mataram's "abdi dalem" — absolutely inappropriate for pre-Islamic setting.
- "Ingwang" is well-attested in Old Sundanese manuscripts (Sanghyang Siksa Kandang Karesian, Bujangga Manik).

## E. Grammar Patterns (Apply Selectively — 30-40% of Sundanese Phrases)

1. **VS Word Order** — Verb before subject in some key phrases
   - "mawa boga kami" instead of "kami mawa boga"
   - Use for formal declarations, prayers, and dramatic moments

2. **-in- infix** — Passive/completed action:
   - ginawé (was made), tinangtu (was determined), pinareng (was permitted)
   - sinembah (was honored), dinarmaan (was gifted), tinulis (was written)

3. **-um- infix** — Active/ongoing action:
   - gumanti (to change/replace), sumebar (scattered/spread)
   - cumeluk (called out), tumuwuh (growing), lumaku (walking/traveling)

4. **ma particle** — Clause marker:
   - "aya ma nu..." (there are those who...)
   - "lamun ma..." (if indeed...)

5. **-keun suffix** — Causative:
   - warahkeun (to teach/convey), tuliskeun (cause to write)
   - hayukeun (to enliven), jayakeun (to make victorious)

6. **mulah** — Prohibition (instead of modern "ulah/tong")
   - "Mulah ragrag!" (Do not fall!)

## H. Revision Rules

1. **Scope:** Only modify Sundanese/Indonesian italicized terms and dialogue in Sundanese context
2. **Do NOT change:** English narration, Javanese dialogue, Sanskrit terms, proper nouns
3. **Fallback:** When no attested Old Sundanese form exists, use the Sanskrit root
4. **Grammar enrichment:** Apply selectively (30-40% of Sundanese phrases) — target formal declarations, prayers, dramatic moments
5. **Emotional register:** Preserve the tone and feeling of each phrase in replacements
6. **Food/drink:** Replace ALL anachronistic items with period-appropriate alternatives
7. **Context preservation:** When replacing terms, ensure surrounding sentences still read naturally
